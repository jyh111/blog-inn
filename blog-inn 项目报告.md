## 2. 使用技术

| 分工   | 详细                                                   |
| :----- | ------------------------------------------------------ |
| 前端   | 主要应用Vue.js，VueX，Vue-router，axios等技术框架      |
| 后端   | 主要使用java语言，基于Spring Boot开源框架，MVC构建实现 |
| 数据库 | 数据层使用Mysql数据库                                  |

## 3. 功能

- 支持用户的注册、登录
- 个人信息查看、修改
- 查看自己的消息，查看收藏的文章
- 博客文章的评论，同时在消息里通知博主
- 博客文章的编辑，增加，修改，删除
- 博客分类及分类的增删改查
- 博客文章的浏览，搜索，评论，收藏
